# 项目简介

## 核心功能
- 用户信息管理：管理员可对用户信息进行统一管理
- 读者信息管理：管理员可对读者信息进行统一管理
- 图书信息管理：管理员可对图书信息进行统一管理
- 借阅记录管理：管理员可对借阅记录进行统一管理
- 图书分类管理：管理员可对图书分类进行统一管理
- 收藏功能：用户和管理员均可使用收藏功能
- 公告发布管理：管理员可发布和管理公告
- 公告查询：用户可查看公告信息
- 借阅记录查询：用户可查看个人借阅记录
- 图书界面：用户可浏览图书信息

## 适用场景
适用于图书馆、学校、社区等需要图书借阅管理的场所，为管理员提供全面的图书借阅管理功能，同时为用户提供便捷的图书查询和借阅服务。

## 技术特点
基于SpringBoot+Vue框架开发，采用协同过滤算法推荐系统，支持图书借阅全流程管理，具备良好的用户体验和系统稳定性。
